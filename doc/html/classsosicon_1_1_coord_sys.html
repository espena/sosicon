Detailed Description

Coordinate system.

Resolves SOSI KOORDSYS codes and converts them to corresponting SRID identifiers if applicable. In SOSI, KOORDSYS is the equivalent to SRID, describing a set of predefined combinations of unit, datum and projection etc.

std::string sosicon::CoordSys::mSrid

SRID.

Spatial reference system identifier, translation of the SOSI KOORDSYS ID. If no SRID exists for current KOORDSYS, this field will be empty. The SRID is a widely adopted standard for referencing various combinations of coordinate system, projection and datum.
